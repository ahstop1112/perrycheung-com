# Perry Cheung — Portfolio Website (2026)

Personal portfolio website showcasing my work as a Senior Frontend / Full Stack Engineer, with a focus on scalable UI architecture, performance, and real-world product experience.

🔗 Live: https://perrycheung.com  
🔗 GitHub: https://github.com/ahstop1112/perrycheung-com

---

## 🧠 Overview

This project is a modern rebuild of my portfolio using a component-driven approach, with emphasis on:

- Clean UI architecture
- Performance optimization
- Maintainable styling system
- Basic end-to-end testing (Playwright)

The goal is not just presentation, but to demonstrate production-level engineering practices.

---

## ⚙️ Tech Stack

- **Framework:** Next.js
- **Language:** TypeScript
- **UI Library:** Material UI (MUI)
- **Styling:** SCSS / Theme-based system
- **i18n:** react-i18next (EN / ZH)
- **Testing:** Playwright (E2E smoke tests)

---

- Portfolio content renders properly
